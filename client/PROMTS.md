## Для отдельного компонента
Оптимизируй и упрости код. Вынеси типы пропсов в отдельный тип type. Название типа обязательно сделай из имени компонента+props. const cn вынести из компонента наверх. Отсортируй импорты в таком порядке 1. вендоры 2. компоненты 3. стили и константы. Между блоками импортов оставить пустую строку. Проверь и исправь через Prettier

## Для отдельного файла стиля
Оптимизируй и упрости код. Отсортируй стили. Проверь и исправь через Prettier
